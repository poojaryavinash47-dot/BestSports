# **App Name**: PlayArena

## Core Features:

- Landing Page: Display a modern, responsive landing page with sections for booking, membership highlights, and testimonials.
- User Authentication: Implement Firebase Authentication for user registration and login using email and Google.
- Cricket Booking System: Allow users to view available cricket grounds, select date and time slots, and book courts.
- Badminton Booking System: Enable users to browse badminton courts, choose dates and times, and complete bookings.
- Membership System: Offer tiered membership plans (Silver, Gold, Platinum) with discounts and priority booking features, managed via Razorpay payments.
- Payment Integration: Integrate Razorpay for processing payments for court bookings and membership subscriptions, storing payment details in Firestore.
- User Dashboard: Provide users with a dashboard to view upcoming bookings, booking history, membership status, and payment history, with options to cancel bookings.

## Style Guidelines:

- Primary color: Neon blue (#08F7FE) to convey a modern sports theme.
- Background color: Dark background (#121212) to create a premium sports platform look with glassmorphism cards.
- Accent color: Neon green (#7CFC00) for highlighting interactive elements and calls to action.
- Body and headline font: 'Space Grotesk' sans-serif for a modern, tech-inspired feel. For longer text, use 'Inter' for body.
- Use sharp, modern icons for sports and booking-related actions.
- Implement a fully responsive layout for optimal viewing on mobile and desktop devices.
- Subtle animations for transitions and user interactions to create a smooth and engaging experience.
